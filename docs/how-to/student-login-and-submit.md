# 学生登录与提交作业

- 适用角色：学生
- 前置条件：学生端可访问；管理员/老师已发放账号信息
- 最后验证日期：2026-02-15

## 登录流程
1. 进入学生端：`http://localhost:3001`。
2. 输入姓名与班级进行身份定位。
3. 使用 `token` 或密码完成认证登录。
4. 首次建议用 token 设置密码，后续可用密码登录。

## 提交流程
1. 打开作业提交入口。
2. 上传答案文件或输入文本内容。
3. 确认提交并等待处理完成。
4. 查看反馈结果与改进建议。

## 验证结果
- 能成功登录并进入学生主界面。
- 提交后可在历史中看到本次提交记录。
- 能查看到反馈内容或状态信息。

## 常见问题
- 同名学生无法定位：补充正确班级。
- token 无效：联系管理员/老师重置。
- 登录后访问受限：检查是否使用了他人 `student_id` 导致越权拦截。
