# 老师日常教学闭环

- 适用角色：老师
- 前置条件：老师端可访问，API 服务健康
- 最后验证日期：2026-02-15

## 目标
完成一次“上传材料 -> 生成分析 -> 复盘调整”的日常闭环。

## 步骤
1. 打开老师端 `http://localhost:3002`。
2. 上传教学相关材料（试卷、作业、课堂记录）。
3. 触发分析生成（讲评、作业建议、知识点薄弱项）。
4. 检查输出质量并进行二次编辑。
5. 保存本次结果作为下一轮教学输入。

## 验证结果
- 至少生成 1 份可读的分析结果。
- 能看到学生或班级层面的改进建议。
- 结果可回溯（在历史记录或相关会话中可找到）。

## 常见失败
- 上传失败：检查文件格式、大小与网络。
- 分析结果空白：检查 API 与 LLM 配置是否可用。
- 结果不稳定：先做小样本测试，再扩大数据量。
