# Teacher Focus Input Schema

## Required
- student_id
- context (lesson/exam/assignment)

## Optional
- notes (free text)
- discussion_notes (md/txt; discussion about recent homework)
- weak_kp (comma/semicolon separated)
- strong_kp
- medium_kp
- next_focus
- answer sheet files (image/PDF)

## Notes
- Use weak/strong KPs if you already know them.
- If only answer sheet is provided, OCR output is saved for traceability.
