# Feynman Reflection (Student-as-Teacher)

Ask the student to explain the learned content **as if teaching a classmate**.

## Prompt Template
"请你用老师的身份，用费曼学习法讲解刚才讨论的核心概念。要求：
- 用自己的话解释，不照抄公式
- 先讲直观图像，再讲公式
- 最后给出一个小例子"

## Evaluation Criteria
- Correctness: no conceptual errors
- Clarity: 3-5 short sentences
- Structure: intuition -> formula -> example

## Follow-up
- If unclear, ask a targeted clarification (single question)
- If correct, proceed to targeted practice
