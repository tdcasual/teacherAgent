# Student Profile (Derived)

This profile is derived from exam responses and practice history. It is safe to update after student interactions.

## Suggested Fields

- student_id
- mastery_by_kp (estimated mastery per knowledge point)
- recent_weak_kp (top 3-5)
- practice_history (recent tasks, completion notes)
- next_focus (what to work on next)
- summary (short readable paragraph)

## Rules

- Do not change raw exam scores or responses.
- Only update derived fields.
- Keep a concise, readable summary for teacher review.
