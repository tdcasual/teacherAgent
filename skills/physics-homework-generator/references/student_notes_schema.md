# Student Notes CSV Schema

## Columns
- student_name (required)
- weak_kp (optional; semicolon or comma separated)
- strength_kp (optional)
- note (optional)
- action (optional; can directly specify assignment text)

## Example
```
student_name,weak_kp,strength_kp,note,action
武熙语, KP-M01;KP-E04, KP-E02, 课堂问答时对电势差方向判断不稳, KP-M01:2基础+1中档; KP-E04:2基础+1中档
```
